# Consulta PLU — React + Vite

Projeto criado a partir da planilha de Carnes e Aves.

## Rodar localmente

```bash
npm install
npm run dev
```

## Gerar versão final

```bash
npm run build
```

## Estrutura

- `src/data/plus.json`: banco local com todos os PLUs importados da planilha.
- `src/utils/calcularDigito.js`: cálculo do último dígito.
- `src/utils/filtros.js`: pesquisa por PLU e categoria.
- `src/App.jsx`: telas de cálculo e pesquisa.
